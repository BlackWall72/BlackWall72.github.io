Bitlocker Status Script

Background: The reason behind this script is:

A)Sometimes the Windows GUI Glitches and won't give you a status while it's encrypting the drive.

B)If you are doing things remote without the desktop, say user is logged in and your using PSTools or something similar, you can't access the GUI

# Use Ctrl + C to Escape # This script MUST be Executed with Administrator Rights! While($true){ manage-bde -status C: ; sleep 5 ; clear ; }

You could also integrate a send keys to keep the pc awake while it's encrypting

# Use Crtl + C to Escape # This Script MUST be Executed with Administrator Rights! $w = New-Object -com WScript.Shell # Load Com Object to Variable While($true){ manage-bde -status C: ; sleep 5 ; $w.SendKeys("{F15}") ; clear ; }
