Raspberry PI arm 64 GUI Fix

So the beta release of the raspberry pi arm 64 OS doesn't offer a light version (disappointing).
as a fix you can create a bash script that can fix that. I've used it and my raspberry works good as new
and as intended (headless)
#!/bin/sh
sudo apt update
sudo apt purge -y x11-common bluez gnome-menus gnome-icon-theme gnome-themes-standard
sudo apt purge -y hicolor-icon-theme gnome-themes-extra-data bluealsa cifs-utils
sudo apt purge -y desktop-base desktop-file-utils
sudo apt autoremove -y
